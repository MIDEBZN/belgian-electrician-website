# Pas d'Électricité dans une Pièce ? Diagnostic et Solutions

Scénario classique : vous êtes dans le salon, tout va bien. Vous allez dans la cuisine : noir complet. Le frigo est éteint, la lumière ne s'allume pas. Pourtant, la télé fonctionne toujours dans la pièce d'à côté. Vous avez une **panne partielle**.

## Étape 1 : Le Tableau Électrique (Check-up)

Allez vérifier votre tableau. Si c'est une panne localisée, le disjoncteur général n'a pas sauté. Cherchez un **petit disjoncteur divisionnaire** qui serait en position basse (OFF).

Si vous en trouvez un : essayez de le remonter.

-   **Il tient :** C'était peut-être une surcharge passagère.
-   **Il ressaute :** Il y a un court-circuit franc sur ce circuit précis. Débranchez tous les appareils de la pièce et réessayez.

## Étape 2 : Le Disjoncteur est ON mais pas de courant

C'est le cas le plus énervant. Tous les disjoncteurs sont levés, mais la pièce est sans électricité. Cela signifie que le circuit est coupé quelque part, mais pas par le disjoncteur.

### La Cause probable : Une connexion desserrée

L'électricité arrive dans la pièce par une première prise ou un interrupteur, puis repart vers les suivantes (repiquage). Si le fil se déconnecte au niveau du premier point, tout le reste de la pièce est coupé.

Il faut ouvrir les prises et les interrupteurs (courant coupé !) pour vérifier si un fil ne s'est pas fait la malle.

## Étape 3 : Le Neutre Coupé

Parfois, vous avez de la tension (si vous testez avec un tournevis testeur, il s'allume), mais les appareils ne fonctionnent pas. C'est le signe que le **fil neutre (bleu)** est coupé quelque part. Le courant arrive mais ne peut pas repartir. C'est une panne vicieuse qui nécessite souvent un électricien pour tracer le câble.
